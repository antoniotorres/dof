# Support

Thanks for using the **Diario Oficial de la Federación** reader! This is an
open-source, community-maintained project, so support is best-effort.

## Where to go

| I want to…                       | Go to                                                                                                  |
| -------------------------------- | ------------------------------------------------------------------------------------------------------ |
| Report a bug                     | [Open a bug report](https://github.com/antoniotorres/dof/issues/new?template=bug_report.yml)           |
| Request a feature                | [Open a feature request](https://github.com/antoniotorres/dof/issues/new?template=feature_request.yml) |
| Ask a question / discuss an idea | [GitHub Discussions](https://github.com/antoniotorres/dof/discussions)                                 |
| Report a security vulnerability  | **Privately** — see [SECURITY.md](SECURITY.md)                                                         |
| Contribute code                  | Read [CONTRIBUTING.md](CONTRIBUTING.md)                                                                |

## Before opening an issue

- **Search first** — your question may already be answered in existing issues
  or discussions.
- Include enough detail to reproduce (URL, steps, browser, screenshots).
- Remember this is **not** the official DOF. For the authoritative gazette,
  consult [dof.gob.mx](https://dof.gob.mx).

## Response expectations

This project is maintained in spare time. Issues and PRs are reviewed as time
allows; thoughtful, well-scoped reports get answered fastest.
